# 02LMM 学习笔记与文档

本目录用于整理 02LMM 项目（Zero to Large Multimodal Model）的学习笔记与博客大纲。
本项目旨在记录从 0 到 LLM，再到原生多模态（LMM）的完整探索过程。整体规划为：

- **基础知识（./基础知识）**：从 0 认识大模型与 Transformer 的核心概念，并与本仓库代码建立对应关系。
- **进阶实践（./进阶实践）**：围绕预训练、指令微调、LoRA、DPO、RLAIF 等完整训练流程，记录更偏工程与调参的实战经验。
- **深度探索（./深度探索）**：针对 MiniMind 的各种小实验，例如不同超参数、不同位置编码、不同推理策略带来的效果差异，以及与上游模型的对比实验。

所有文档均基于 Vibe Writing 大模型输出结果，在此基础上由我进行整理、裁剪与校订，作为个人学习记录与公开博客使用；同时尽量标注与本仓库中具体脚本 / 模块的关联，方便一边阅读一边看代码。

> 提示：当前优先完善「基础知识」模块，后续可以在本文件中扩展其他模块的说明与导航。
